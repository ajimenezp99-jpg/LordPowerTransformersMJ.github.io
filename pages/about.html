▸ DESCRIPCIÓN DEL SISTEMA
SGM · TRANSPOWER es una plataforma web integral desarrollada para la dirección y gestión del mantenimiento especializado de transformadores de potencia en servicio activo. El sistema opera bajo un esquema de ciclo de vida completo: desde el registro del activo hasta el cierre de intervenciones y análisis de causa raíz.

⚙️ Objetivo Principal

Garantizar la trazabilidad completa de todas las intervenciones de mantenimiento preventivo y correctivo sobre transformadores de potencia, alineado con los estándares internacionales IEEE C57.12, IEC 60076, la normativa colombiana RETIE y el sistema de gestión de energía ISO 50001:2018.

🎯 Enfoque Técnico

  • Indicadores RAM: confiabilidad, disponibilidad y mantenibilidad
  • Análisis de causa raíz (RCA) por falla
  • Historial inmutable de intervenciones
  • Gestión documental normativa y auditable
  • Alertas tempranas por vencimiento de mantenimientos
  • Visualización georreferenciada de activos
▸ NATURALEZA DEL PROYECTO

🤝 Sin Ánimo de Lucro

El sistema se diseña y opera íntegramente sobre niveles gratuitos (tier free) de proveedores cloud: GitHub Pages, Vercel Hobby, Firebase Spark y Leaflet/OpenStreetMap. No hay costos de operación previstos en las fases actuales.

🔒 Acceso Controlado

La plataforma incorpora una barrera de acceso estática durante la fase de construcción, que será reemplazada por autenticación Firebase Auth en la Fase 5. El panel administrativo contará con credenciales independientes y mayor nivel de protección.

📐 Stack Tecnológico

  • Frontend: HTML5, CSS3, JavaScript ES6+
  • Hosting: Vercel + GitHub Pages
  • Auth: Firebase Authentication
  • Base de datos: Cloud Firestore
  • Almacenamiento: Firebase Storage
  • Mapas: Leaflet + OpenStreetMap

📅 Hoja de Ruta

El desarrollo sigue 15 microfases iterativas (F0–F14), cada una cerrada con commit aislado. Avance actual: 20 %. Las próximas fases incluyen la integración de Firebase, los módulos operativos (inventario, órdenes, KPIs, georreferenciación) y el gate dinámico.
