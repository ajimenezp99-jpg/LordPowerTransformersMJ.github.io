▸ PUESTA EN MARCHA v2

Pasos de bootstrap recomendados para arrancar el sistema:

  1. Catálogos §A7 → click CARGAR BASELINE OFICIAL. Crea las 31 subactividades, 7 macroactividades y 12 causantes.
  2. Usuarios → alta de los miembros del equipo con rol F28 (director_proyectos, analista_tx, gestor_contractual, brigadista, auditor_campo) + zonas[] de ámbito.
  3. Contratos → registro de los 8 contratos macro vigentes con presupuesto comprometido / ejecutado.
  4. Subestaciones → registro de subestaciones (FK que apuntará desde cada TX).
  5. Importador → carga del Excel oficial 296 TX. SIMULAR primero, revisar discrepancias, luego IMPORTAR.
  6. Umbrales → confirmar/ajustar baselines MO.00418 si el Profesional de Tx solicita una calibración local.
  7. Alertas → activar cron F32 con email del director (requiere despliegue previo de Cloud Functions, ver docs/DEPLOY-FUNCTIONS.md).
